#### 13강. 문서의 구조와 슈퍼스타들

```html
<title>제목</title>
title은 책 표지와 같이 인지하므로 검색 등에 반영된다. 따라서 title을 쓰는 것이 좋다. 정보가 있어도 검색에 노출이 안되면 의미가 없습니다!
<meta charset="utf-8"> 캐릭터셋을 utf-8 방식으로 저장하였다.
```

html구조(외우기! 제로 베이스에서 칠 수 있을 만큼!)

```html
<!doctype html>
<html>
  <head>
    <title></title>
    <meta charset="utf-8">
  </head>
  <body>
  </body>
</html>
```
